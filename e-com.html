Creating an E-commerce Site with React and Node.js

An online store built with React and Node.js.

Project Overview

Title: E-commerce Site

Description: An online store built with React and Node.js.

Details: Build a fully functional e-commerce site with product listings, a shopping cart, and a checkout system. Technologies used include React for the frontend and Node.js for the backend.

Step 1: Setting Up the Project

Frontend Setup with React


  1. Create a new React application:

    2.                         npx create-react-app ecommerce-site
    cd ecommerce-site

  2. Install necessary dependencies:

    3.                         npm install react-router-dom axios

Backend Setup with Node.js


  1. Create a new directory for the backend:

    2.                         mkdir backend
    cd backend
    npm init -y

  2. Install necessary dependencies:

    3.                         npm install express mongoose cors dotenv

import React, { useState, useEffect } from 'react';
import axios from 'axios';

function ProductList() {
  const [products, setProducts] = useState([]);

  useEffect(() => {

Step 3: Creating the Backend


Setting Up Express and MongoDB



  1. Create the necessary folders and files:

    2.                         mkdir models routes
    touch index.js models/Product.js routes/products.js

  2. Connect to MongoDB and set up Express server in index.js:

    3.                         
                            {`
const express = require('express');
const mongoose = require('mongoose');
const cors = require('cors');
const dotenv = require('dotenv');

dotenv.config();

const app = express();
const PORT = process.env.PORT || 5000;

app.use(cors());
app.use(express.json());

mongoose.connect(process.env.MONGO_URI, {
  useNewUrlParser: true,
  useUnifiedTopology: true,
}).then(() => console.log('MongoDB connected'))
  .catch(error => console.error('MongoDB connection error:', error));

app.use('/api/products', require('./routes/products'));

app.listen(PORT, () => console.log(\`Server running on port \${PORT}\`));
                            `}

Creating the Product Model


  1. In models/Product.js:

    2.                         
                            {`
const mongoose = require('mongoose');

const ProductSchema = new mongoose.Schema({
  name: {
    type: String,
    required: true,
  },
  description: {
    type: String,
    required: true,
  },
  price: {
    type: Number,
    required: true,
  },
});

module.exports = mongoose.model('Product', ProductSchema);
                            `}

Creating the Products Route


  1. In routes/products.js:

    2.                         
                            {`
const express = require('express');
const Product = require('../models/Product');
const router = express.Router();

// Get all products
router.get('/', async (req, res) => {
  try {
    const products = await Product.find();
    res.json(products);
  } catch (error) {
    res.status(500).json({ message: error.message });
  }
});

// Add a new product
router.post('/', async (req, res) => {
  const product = new Product({
    name: req.body.name,
    description: req.body.description,
    price: req.body.price,
  });

  try {
    const newProduct = await product.save();
    res.status(201).json(newProduct);
  } catch (error) {
    res.status(400).json({ message: error.message });
  }
});

module.exports = router;
                            `}

Step 4: Testing and Running the Application

  1. Start the backend server:
    2.                         node index.js
  2. Start the React frontend:
    3.                         npm start
