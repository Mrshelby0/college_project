Creating a Stunning Portfolio Using HTML, CSS, and JavaScript

By Sparsh Sharma

Step 1: Setting Up the Project

Create a new directory for your project. Inside this directory, create three files: index.html, styles.css, and script.js.

    portfolio/
    |-- index.html
    |-- styles.css
    |-- script.js

Step 2: Creating the HTML Structure

Open index.html and set up the basic HTML structure.

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Portfolio</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <header>
            <h1>Sparsh Sharma</h1>
            <nav>
                <ul>
                    <li><a href="#about">About Me</a></li>
                    <li><a href="#projects">Projects</a></li>
                    <li><a href="#contact">Contact</a></li>
                </ul>
            </nav>
        </header>
    
        <section id="about">
            <h2>About Me</h2>
            <p>Hello! I'm Sparsh Sharma, a computer application student and blog
             writer with 2 years of experience...</p>
        </section>
    
        <section id="projects">
            <h2>Projects</h2>
            <div class="project">
                <h3>Project 1</h3>
                <p>Description of project 1.</p>
            </div>
            <div class="project">
                <h3>Project 2</h3>
                <p>Description of project 2.</p>
            </div>
        </section>
    
        <section id="contact">
            <h2>Contact</h2>
            <p>You can reach me at: <a href="mailto:example@example.com">
            example@example.com</a></p>
        </section>
    
        <script src="script.js"></script>
    </body>
    </html>

Step 3: Styling with CSS

Open styles.css and add styles to make your portfolio visually appealing. We will use your preferred theme colors: black and #5cb9a6.

    body {
        font-family: Arial, sans-serif;
        margin: 0;
        padding: 0;
        background-color: #f4f4f4;
        color: #333;
    }
    
    header {
        background-color: black;
        color: white;
        padding: 20px 0;
        text-align: center;
    }
    
    header nav ul {
        list-style: none;
        padding: 0;
    }
    
    header nav ul li {
        display: inline;
        margin: 0 15px;
    }
    
    header nav ul li a {
        color: #5cb9a6;
        text-decoration: none;
    }
    
    section {
        padding: 20px;
        margin: 20px;
        background-color: white;
        border-radius: 5px;
        box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
    }
    
    h2 {
        color: #5cb9a6;
    }
    
    .project {
        margin-bottom: 20px;
    }

Step 4: Adding Interactivity with JavaScript

Open script.js and add JavaScript to enhance user experience. For this example, we'll add a smooth scroll effect when navigating through the sections.

    document.addEventListener('DOMContentLoaded', function () {
        const links = document.querySelectorAll('nav ul li a');
    
        for (const link of links) {
            link.addEventListener('click', smoothScroll);
        }
    
        function smoothScroll(event) {
            event.preventDefault();
            const targetId = event.currentTarget.getAttribute('href').substring(1);
            const targetElement = document.getElementById(targetId);
    
            window.scrollTo({
                top: targetElement.offsetTop,
                behavior: 'smooth'
            });
        }
    });

Step 5: Testing Your Portfolio

Open index.html in your web browser to see your portfolio in action. Test the navigation links to ensure the smooth scroll effect works as expected.

Conclusion

Congratulations! You've created a simple and elegant portfolio using HTML, CSS, and JavaScript. This portfolio serves as a foundation that you can expand upon by adding more projects, improving styles, and incorporating advanced features. A well-designed portfolio is a powerful tool to showcase your skills and make a strong impression on potential employers or clients.

Feel free to customize the content, style, and functionality to match your personal brand and preferences. Happy coding!
